# Backend Challenge - Observability Systems

## Introduction

Implement comprehensive observability with metrics, logs, and traces.

This challenge is tech-agnostic. Choose your preferred programming language, framework, database, ORM, and observability tools. For senior level, you can use any technologies and patterns.

### Objectives

- Set up metrics collection and visualization
- Implement structured logging
- Configure distributed tracing

### Instructions

1. **Environment Setup**:
    - Programming language and web framework
    - Database and ORM
    - Metrics collection (Prometheus, etc.)
    - Logging aggregation (ELK stack, etc.)
    - Tracing system (Jaeger, Zipkin, etc.)

2. **Implementation Details**:
    - Instrument application with metrics
    - Implement structured logging
    - Set up distributed tracing across services

3. **Testing**:
    - Generate and analyze observability data

### Possible Improvements

- Add custom dashboards
- Implement alerting rules
- Set up log correlation

## Conclusion

By completing this challenge, you will master system observability.

Happy coding!
