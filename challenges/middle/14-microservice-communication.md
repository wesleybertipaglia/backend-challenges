# Backend Challenge - Microservice Communication

## Introduction

Implement communication between microservices using message queues.

This challenge is tech-agnostic. Choose your preferred programming language, framework, database, ORM, message queue, and caching solution. For middle level, you can use advanced features like caching.

### Objectives

- Set up inter-service communication
- Handle service discovery
- Implement event-driven architecture

### Instructions

1. **Environment Setup**:
    - Programming language and web framework
    - Database and ORM
    - Message queue system
    - Service discovery tool
    - Caching solution
    - Docker (recommended for containerization)

2. **Implementation Details**:
    - Create multiple microservices
    - Implement message publishing and consumption
    - Handle service-to-service calls

3. **Testing**:
    - Test inter-service communication

### Possible Improvements

- Add circuit breakers
- Implement distributed tracing

## Conclusion

By completing this challenge, you will understand microservice patterns.

Happy coding!
