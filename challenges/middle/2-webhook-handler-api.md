# Backend Challenge - Webhook Handler API

## Introduction

Create an API that receives and processes webhooks from external services.

This challenge is tech-agnostic. Choose your preferred programming language, framework, database, ORM, and caching solution. For middle level, you can use advanced features like caching.

### Objectives

- Handle incoming webhook requests
- Validate webhook signatures
- Process and store webhook data

### Instructions

1. **Environment Setup**:
    - Programming language and web framework
    - Database and ORM
    - Caching solution
    - Docker (recommended for containerization)

2. **Implementation Details**:
    - Create a POST endpoint to receive webhooks
    - Implement signature verification
    - Store and process webhook payloads

3. **Testing**:
    - Send test webhooks and verify processing

### Possible Improvements

- Add retry logic for failed deliveries
- Implement webhook queuing

## Conclusion

By completing this challenge, you will understand webhook integration and security.

Happy coding!
