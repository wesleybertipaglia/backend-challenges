# Backend Challenge - Word Counter API

## Introduction

Build a web service that counts words, characters, and other metrics in a given text.

This challenge is tech-agnostic. Choose your preferred programming language and framework. For beginner level, no database required.

### Objectives

- Process text input
- Perform counting operations
- Return detailed statistics

### Instructions

1. **Environment Setup**:
    - Programming language and web framework

2. **Implementation Details**:
    - Create a POST endpoint at `/count` that accepts text in the request body
    - Return counts for words, characters, sentences, etc.

3. **Testing**:
    - Test with POST request containing text

### Possible Improvements

- Exclude common words (stop words)
- Provide readability scores
- Support different languages

## Conclusion

By completing this challenge, you will learn text processing and analytics in web services.

Happy coding!
