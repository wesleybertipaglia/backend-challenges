# Backend Challenge - Weather API

## Introduction

Create a simple weather service that returns mock weather data for a given city.

This challenge is tech-agnostic. Choose your preferred programming language and framework. For beginner level, no database required.

### Objectives

- Simulate external API responses
- Handle different request parameters

### Instructions

1. **Environment Setup**:
    - Programming language and web framework

2. **Implementation Details**:
    - Create a GET endpoint at `/weather` that accepts a `city` parameter and returns mock weather data like temperature and condition

3. **Testing**:
    - Test with: `GET /weather?city=London`

### Possible Improvements

- Add more weather details
- Implement error handling for unknown cities

## Conclusion

By completing this challenge, you will practice building mock APIs.

Happy coding!
