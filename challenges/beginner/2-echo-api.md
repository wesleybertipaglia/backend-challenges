# Backend Challenge - Echo API

## Introduction

Build a web service that echoes back the input provided by the user.

This challenge is tech-agnostic. Choose your preferred programming language and framework. For beginner level, no database required.

### Objectives

- Handle request parameters
- Return dynamic responses based on input

### Instructions

1. **Environment Setup**:
    - Programming language and web framework

2. **Implementation Details**:
    - Create a GET endpoint at `/echo` that accepts a query parameter `message` and returns it in the response: `{"echo": "your message"}`

3. **Testing**:
    - Test with: `GET /echo?message=hello`

### Possible Improvements

- Support POST requests with JSON body
- Add validation for input length

## Conclusion

By completing this challenge, you will learn to handle and process user input in web services.

Happy coding!
