# Backend Challenge - Chat Application API

## Introduction

Create an API for a real-time chat application with rooms and direct messages.

This challenge is tech-agnostic. Choose your preferred programming language, framework, database, and ORM. For junior level, focus on core technologies without advanced features like caching.

### Objectives

- Handle real-time messaging
- Manage chat rooms and users
- Store message history

### Instructions

1. **Environment Setup**:
    - Programming language and web framework
    - Real-time communication library
    - Database and ORM
    - Docker (recommended for containerization)
    
2. **Implementation Details**: 
    - Create endpoints for rooms, messages, and users
    - Implement real-time communication for chat

3. **Testing**: 
    - Connect clients and exchange messages

### Possible Improvements

- Add file sharing
- Implement typing indicators
- Add message encryption

## Conclusion

By completing this challenge, you will gain experience with real-time applications.

Happy coding!
