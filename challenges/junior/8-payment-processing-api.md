# Backend Challenge - Payment Processing API

## Introduction

Develop an API to handle payments, process transactions, and manage payment methods.

This challenge is tech-agnostic. Choose your preferred programming language, framework, database, and ORM. For junior level, focus on core technologies without advanced features like caching.

### Objectives

- Integrate with payment gateways
- Handle secure transactions
- Manage refunds and disputes

### Instructions

1. **Environment Setup**:
    - Programming language and web framework
    - Docker (recommended for containerization)
    
2. **Implementation Details**: 
    - Create endpoints for payment creation, confirmation, and refunds
    - Integrate with a payment gateway

3. **Testing**: 
    - Process test payments

### Possible Improvements

- Add subscription billing
- Implement webhooks
- Support multiple currencies

## Conclusion

By completing this challenge, you will learn payment integration and financial APIs.

Happy coding!
