# Backend Challenge - URL Shortener API

## Introduction

Build an API that shortens URLs and redirects to the original URLs.

This challenge is tech-agnostic. Choose your preferred programming language, framework, database, and ORM. For junior level, focus on core technologies without advanced features like caching.

### Objectives

- Generate short codes for URLs
- Handle redirects
- Track usage statistics

### Instructions

1. **Environment Setup**:
    - Programming language and web framework
    - Database and ORM
    - Docker (recommended for containerization)
    
2. **Implementation Details**: 
    - Create endpoints: POST /shorten (to create short URL), GET /:code (to redirect)
    - Store original URLs and short codes in database

3. **Testing**: 
    - Shorten a URL and test the redirect

### Possible Improvements

- Add custom short codes
- Implement expiration dates
- Provide analytics on clicks

## Conclusion

By completing this challenge, you will learn URL manipulation and data storage.

Happy coding!
