# Backend Challenge - Notification API

## Introduction

Build an API for sending notifications via email, SMS, and push notifications.

This challenge is tech-agnostic. Choose your preferred programming language, framework, database, and ORM. For junior level, focus on core technologies without advanced features like caching.

### Objectives

- Support multiple notification channels
- Manage user preferences
- Handle delivery status

### Instructions

1. **Environment Setup**:
    - Programming language and web framework
    - Database and ORM
    - Docker (recommended for containerization)
    
2. **Implementation Details**: 
    - Create endpoints to send notifications and manage subscriptions
    - Integrate with notification service providers

3. **Testing**: 
    - Send notifications and check delivery

### Possible Improvements

- Add scheduling
- Implement templates
- Provide analytics

## Conclusion

By completing this challenge, you will understand multi-channel communication systems.

Happy coding!
